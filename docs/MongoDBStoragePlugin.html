MongoDBStoragePlugin

MongoDBStoragePlugin

A storage plugin backed by MongoDB. This plugin is not built into Monochrome, it must be installed separately. npm install @monochrome-bot/mongodb-storage-plugin

Instantiate a plugin. A collection called monochrome will be created in the specified database. All of monochrome's internal data will be stored there.
