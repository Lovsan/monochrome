NavigationManager

NavigationManager

Handles displaying and controlling Navigations. The NavigationManager can be accessed via Monochrome#getNavigationManager. For an example of creating a navigation, see the demo navigation command.

Methods

show(navigation, expirationTimeInMs, channel, parentMsgopt)

Source:
  • navigation_manager.js, line 26
Display a navigation.
Parameters:
Name Type Attributes Description
navigation Navigation The navigation to display.
expirationTimeInMs number How long before the navigation should become inactive and cease responding to reactions. Note that showing a navigation takes memory, so if this number is excessively high, you may run out of memory eventually (unlikely to ever happen except for very popular bots).
channel external:"Eris.Channel" The channel to show the navigation in (Eris.Message has a .channel property to get this)
parentMsg external:"Eris.Message" <optional>
 The user message that caused the navigation to be created. If that message is deleted, the bot will delete the navigation. Omit if you don't want that behavior.
